## Building

First install the following:

* mkdocs
* mkdocs-material - python package

Then simply run `mkdocs serve` to host a local version of the docs.

## French doc

Serve:

`mkdocs serve -f mkdocs-fr.yml`

Build:
`mkdocs build -f mkdocs-fr.yml -d site-fr`
